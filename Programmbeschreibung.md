# Fleet Navigator - Programmbeschreibung

## Ihr persönlicher KI-Assistent. Lokal. Privat. Kostenlos.

**Fleet Navigator** ist eine professionelle Desktop-Anwendung, die modernste KI-Technologie direkt auf Ihren Computer bringt - ohne Cloud, ohne Abonnement, ohne Datenweitergabe.

---

## Die wichtigsten Vorteile auf einen Blick

### 100% Privatsphäre
Ihre Daten verlassen niemals Ihren Computer. Keine Cloud-Anbindung, keine Telemetrie, keine versteckten Uploads. Was Sie mit Fleet Navigator besprechen, bleibt bei Ihnen.

### Keine laufenden Kosten
Einmal installiert, für immer nutzbar. Keine monatlichen Gebühren, keine Token-Limits, keine versteckten Kosten. Die KI läuft vollständig auf Ihrer Hardware.

### Funktioniert offline
Nach der ersten Einrichtung benötigt Fleet Navigator keine Internetverbindung mehr. Ideal für sensible Arbeitsumgebungen oder unterwegs.

---

## Kernfunktionen

### Intelligenter Chat-Assistent
- Natürliche Gespräche mit modernster KI-Technologie
- Mehrere parallele Unterhaltungen verwalten
- Vollständiger Gesprächsverlauf wird automatisch gespeichert
- Antworten erscheinen in Echtzeit (Streaming)

### Experten-System mit Persönlichkeiten
Erstellen Sie spezialisierte KI-Assistenten für verschiedene Aufgaben:
- **Rechtsberater** für juristische Formulierungen
- **Steuerberater** für Finanzfragen
- **Programmierer** für Code-Unterstützung
- **Texter** für kreative Inhalte
- Eigene Experten mit individuellen "Blickwinkeln" definierbar

### Professionelle Brieferstellung
- KI-generierte Briefe im **DIN 5008 Format**
- Deutsche und englische Vorlagen für häufige Anlässe
- Export als **DOCX** (Word) oder **ODT** (LibreOffice)
- Automatische Übernahme Ihrer persönlichen Absenderdaten

### Dokumenten-Analyse (RAG)
- **PDF, Word, ODT, TXT** und weitere Formate durchsuchbar
- Fragen Sie die KI zu Ihren Dokumenten
- Volltext-Indexierung für blitzschnelle Suche
- Web-Recherche mit KI-optimierten Suchanfragen

### Bildverständnis (Vision)
- Laden Sie Bilder hoch und lassen Sie sie analysieren
- Automatische Texterkennung (OCR) für eingescannte Dokumente
- Unterstützt PNG, JPG, WebP und weitere Formate

---

## Technische Highlights

### Leistungsstarke KI-Engine
- **GPU-Beschleunigung** für NVIDIA-Grafikkarten (CUDA)
- **CPU-Modus** für Systeme ohne dedizierte Grafikkarte
- Unterstützung für über **100 verschiedene KI-Modelle**
- Integrierter **Model Store** zum direkten Download von Modellen

### Moderne Benutzeroberfläche
- Übersichtliches, deutschsprachiges Interface
- Responsive Design für verschiedene Bildschirmgrößen
- Helles und dunkles Farbschema
- Keine Einarbeitung nötig - sofort loslegen

### Plattformübergreifend
- **Windows** (10/11)
- **Linux** (Ubuntu, Debian, etc.)
- **macOS** (Intel & Apple Silicon)

## Fleet Mates - Ihr KI-Netzwerk

Verbinden Sie mehrere Geräte zu einem intelligenten Netzwerk:
- **Sicheres Pairing** mit Ende-zu-Ende-Verschlüsselung
- System-Überwachung entfernter Computer
- Dateizugriff über das Netzwerk
- Terminal-Befehle aus der Ferne ausführen

## Installation

Eine einzige Datei - mehr braucht es nicht:

```bash
java -jar fleet-navigator.jar
```

Öffnen Sie anschließend `http://localhost:2025` im Browser.
